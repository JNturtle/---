小狼毫輸入法的臺灣注音輸入方案
※需要先安裝「RIME|中州韻輸入法引擎」，然後直接套用本設定，重新部署與關閉重啟輸入法，讓輸入方案能運作。

因為「注音」輸入方案十分少見，於是想弄個微軟注音習慣的人可以好上手的輸入方案。

如何將輸入方案套上？
點擊「Code」，「Download ZIP」，下載完成後解壓縮，將「taiwan_zhuyin」內的檔案貼上到「用戶資料夾」，重新部署，關閉輸入法，啟動輸入法。
